== Description ==

This Plugin is a very simple Plugin and allows you to add with the shortcode [mybcn] a breadcrumb Navigation to your Page ore Post.

== Frequently Asked Questions ==

=How can i use it?=

Just activate the plugin and put the shortcode [mybcn] where you want the breadcrumb to be display.
